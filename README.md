# Radio Streamer

![RPI3](https://github.com/user-attachments/assets/7f7e7a1a-5c35-44b1-9fc7-c8023e928f5e)<br><br>

Software used for this project:
1) Raspberry Pi OS Lite, a port of Debian with no desktop environment. I used a Raspberry Pi due to its low consumption but you can set it up on any Debian-compatible system.
2) FFMPEG, UFW, SSH, RFKILL, LSHW, WPASUPPLICANT, RESOLVCONF, and some custom Bash Scripts.
